1. Make sure to install python, pycharm and libraries that are mentioned in Rrequirement.txt file
and setup the virtual env
2. Chrome or mozilla must the installed in your pc
3. For execution use below command:
pytest --alluredir= reports/<any name> -v -s test_Make_My_Trip.py
The execution is performed and report files are stored in Reports directory,
fetch the path and run command > "allure serve <report path>" to generate report.
